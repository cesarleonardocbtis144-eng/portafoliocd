Módulo 01 · Introducción

¿Qué es CSS3?

CSS (Cascading Style Sheets) es el lenguaje que da apariencia visual a las páginas web. Mientras HTML define la estructura, CSS define los colores, tamaños, posiciones, fuentes, animaciones y todo lo que ves visualmente.

Se llama Cascading (en cascada) porque los estilos se heredan y se sobreescriben según reglas de especificidad y orden.

Las tres formas de incluir CSS

formas-de-incluir.html
/* 1. EXTERNO (recomendado) — enlazado en el <head> */ <link rel="stylesheet" href="styles.css"/> /* 2. INTERNO — dentro del <head> en una etiqueta <style> */ <style> p { color: red; } </style> /* 3. EN LÍNEA — directamente en el elemento (evitar) */ <p style="color: red; font-size: 18px;">Texto</p>

Anatomía de una regla CSS

anatomia.css
/* selector { propiedad: valor; } */ h1 { /* selector */ color: #e8c97a; /* propiedad: valor */ font-size: 2rem; font-weight: bold; margin-bottom: 16px; } /* fin de la regla */ /* Múltiples selectores con la misma regla */ h1, h2, h3 { font-family: 'Cormorant Garamond', serif; line-height: 1.2; }
La cascada CSS: cuando varios estilos se aplican al mismo elemento, gana el que tenga mayor especificidad. El orden es: !important > estilos en línea > ID > clase > etiqueta. En caso de empate, gana el último declarado.

Módulo 02 · Targeting

Selectores CSS

Los selectores son la forma en que CSS apunta a los elementos HTML que quiere estilizar. Dominarlos es la clave para escribir CSS eficiente.

Selectores básicos y combinadores

selectores.css
/* ── BÁSICOS ── */ * { box-sizing: border-box; } /* universal */ p { color: white; } /* etiqueta */ .card { padding: 20px; } /* clase */ #menu { position: fixed; } /* ID */ /* ── COMBINADORES ── */ nav a { color: gold; } /* descendiente */ nav > a { font-weight: bold; } /* hijo directo */ h1 + p { margin-top: 0; } /* hermano adyacente */ h1 ~ p { opacity: 0.8; } /* hermanos siguientes */ /* ── ATRIBUTOS ── */ a[href] { color: blue; } /* tiene atributo */ a[href^="https"] { color: green; } /* empieza con */ a[href$=".pdf"] { color: red; } /* termina con */ img[alt*="logo"] { width: 100px; } /* contiene */ /* ── PSEUDOCLASES ── */ a:hover { text-decoration: underline; } a:focus { outline: 2px solid gold; } button:active { transform: scale(0.97); } input:disabled { opacity: 0.5; } li:first-child { font-weight: bold; } li:last-child { border-bottom: none; } li:nth-child(2n){ background: rgba(255,255,255,0.05); } p:not(.intro) { color: gray; } /* ── PSEUDOELEMENTOS ── */ p::first-line { font-weight: bold; } p::first-letter{ font-size: 2em; float: left; } .card::before { content: '✦'; color: gold; } .card::after { content: ''; display: block; } ::selection { background: gold; color: black; }

Módulo 03 · Espaciado y tamaño

Modelo de Caja

Todo elemento HTML es una caja rectangular. Entender cómo se calcula su tamaño y espacio es fundamental para dominar CSS.

Vista visual
margin
border
padding
content
box-model.css
.caja { /* TAMAÑO */ width: 300px; height: 200px; min-width: 200px; max-width: 100%; min-height: 100px; max-height: 500px; /* PADDING — espacio interior (4 valores: top right bottom left) */ padding: 20px; /* todos los lados */ padding: 10px 20px; /* vertical | horizontal */ padding: 10px 20px 15px 20px; /* top right bottom left */ padding-top: 10px; /* BORDER */ border: 1px solid gold; border-radius: 12px; /* esquinas redondeadas */ border-top: 2px dashed white; /* MARGIN — espacio exterior */ margin: 20px; margin: 0 auto; /* centrar horizontalmente */ margin-bottom: 32px; /* box-sizing: include border+padding en el width/height */ box-sizing: border-box; /* SIEMPRE usar esto */ /* OVERFLOW */ overflow: hidden; /* hidden | scroll | auto */ overflow-x: auto; }
Regla de oro: Siempre agrega *, *::before, *::after { box-sizing: border-box; } al inicio de tu CSS. Esto hace que el padding y border se incluyan dentro del width, evitando confusiones de tamaño.

Módulo 04 · Tipografía web

Texto & Fuentes

CSS3 tiene decenas de propiedades para controlar la tipografía. Usadas correctamente transforman por completo la legibilidad y personalidad visual de un sitio.

tipografia.css
body { /* FUENTE */ font-family: 'DM Sans', sans-serif; /* fuente principal, fallback */ font-size: 16px; /* tamaño base */ font-weight: 400; /* 100–900 o bold/normal */ font-style: normal; /* italic | oblique */ /* TEXTO */ line-height: 1.6; /* interlineado */ letter-spacing: 0.5px; /* espacio entre letras */ word-spacing: 2px; /* espacio entre palabras */ text-align: left; /* left | center | right | justify */ text-transform: none; /* uppercase | lowercase | capitalize */ text-decoration: none; /* underline | line-through | overline */ text-shadow: 0 2px 8px rgba(0,0,0,0.3); white-space: nowrap; /* nowrap | pre | pre-wrap */ word-break: break-word; } /* FUENTES VARIABLES (CSS3 moderno) */ h1 { font-size: clamp(2rem, 5vw, 4rem); /* mín, ideal, máx */ font-variation-settings: "wght" 700; } /* CARGAR FUENTES EXTERNAS */ @font-face { font-family: 'MiFuente'; src: url('fuente.woff2') format('woff2'); font-display: swap; }
font-family
Define la fuente. Siempre incluir un fallback genérico.
font-size
Tamaño. Usa rem para accesibilidad, em para relativo.
font-weight
Grosor: 100 (thin) a 900 (black). Normal=400, Bold=700.
line-height
Interlineado. 1.5–1.8 es óptimo para cuerpo de texto.
letter-spacing
Espacio entre caracteres. Positivo separa, negativo une.
text-transform
Convierte a mayúsculas, minúsculas o capitalize.
text-shadow
Sombra de texto: x y blur color.
clamp()
Tamaño fluido entre un mínimo y máximo según viewport.

flexbox.css
/* ── CONTENEDOR FLEX ── */ .contenedor { display: flex; /* activa flexbox */ flex-direction: row; /* row | row-reverse | column | column-reverse */ flex-wrap: wrap; /* nowrap | wrap | wrap-reverse */ gap: 16px; /* espacio entre items */ column-gap: 20px; /* solo entre columnas */ row-gap: 12px; /* solo entre filas */ /* Alineación en el eje principal (horizontal en row) */ justify-content: space-between; /* flex-start | center | space-around | space-evenly */ /* Alineación en el eje cruzado (vertical en row) */ align-items: center; /* flex-start | flex-end | stretch | baseline */ /* Alineación de múltiples líneas (cuando flex-wrap) */ align-content: flex-start; } /* ── ITEMS FLEX ── */ .item { flex: 1; /* crecer, encoger y base: shorthand */ flex-grow: 1; /* cuánto crece relativo a otros */ flex-shrink: 1; /* cuánto se encoge si no hay espacio */ flex-basis: auto; /* tamaño base antes de crecer/encoger */ align-self: flex-end; /* sobreescribe align-items para este item */ order: 2; /* reordena visualmente (default 0) */ } /* CENTRADO PERFECTO — lo más fácil del mundo */ .centrado { display: flex; justify-content: center; align-items: center; }

Módulo 08 · Posición y capas

Posicionamiento

CSS ofrece diferentes modos de posicionar elementos fuera del flujo normal del documento, desde posición relativa hasta fija en pantalla.

posicionamiento.css
.elemento { /* TIPOS DE POSICIÓN */ position: static; /* default — flujo normal */ position: relative; /* se mueve relativo a sí mismo */ position: absolute; /* relativo al padre con position != static */ position: fixed; /* fijo en la pantalla (no se mueve al scroll) */ position: sticky; /* sigue el scroll hasta un punto */ /* COORDENADAS (con position != static) */ top: 20px; right: 0; bottom: 0; left: 50%; /* CENTRAR con absolute */ position: absolute; top: 50%; left: 50%; transform: translate(-50%, -50%); /* CAPAS */ z-index: 10; /* mayor número = más arriba */ /* DISPLAY */ display: block; /* block | inline | inline-block | none */ visibility: hidden; /* oculta pero ocupa espacio */ } /* STICKY: se pega al top al hacer scroll */ .navbar { position: sticky; top: 0; z-index: 100; }

Módulo 09 · CSS moderno

Variables & Custom Properties

Las variables CSS (también llamadas Custom Properties) permiten definir valores reutilizables en toda la hoja de estilos. Son una de las características más poderosas de CSS3 moderno.

variables.css
/* Definir variables en :root (disponibles en toda la página) */ :root { /* Colores */ --color-primario: #c9a84c; --color-fondo: #07111f; --color-texto: #f5f8ff; --color-sutil: rgba(245, 248, 255, 0.5); /* Tipografía */ --fuente-titulo: 'Cormorant Garamond', serif; --fuente-cuerpo: 'DM Sans', sans-serif; --fuente-codigo: 'JetBrains Mono', monospace; /* Espaciado */ --espacio-xs: 4px; --espacio-sm: 8px; --espacio-md: 16px; --espacio-lg: 32px; --espacio-xl: 64px; /* Radios */ --radio-sm: 6px; --radio-md: 12px; --radio-lg: 18px; /* Transiciones */ --trans-rapida: all 0.2s ease; --trans-suave: all 0.4s cubic-bezier(.22, 1, .36, 1); } /* Usar variables con var() */ .tarjeta { background: var(--color-fondo); color: var(--color-texto); font-family: var(--fuente-cuerpo); padding: var(--espacio-md); border-radius: var(--radio-lg); transition: var(--trans-suave); /* var() con valor de respaldo */ border: 1px solid var(--color-borde, rgba(255,255,255,0.1)); } /* Tema oscuro / claro con variables */ @media (prefers-color-scheme: dark) { :root { --color-fondo: #07111f; --color-texto: #f5f8ff; } } /* Modificar variables localmente */ .tarjeta-dorada { --color-primario: #e8c97a; /* solo afecta a esta tarjeta */ }

Módulo 10 · Movimiento

Transiciones & Animaciones

CSS3 permite crear movimiento y transiciones suaves sin necesidad de JavaScript. Las transiciones son cambios entre dos estados, las animaciones son secuencias completas.

animaciones.css
/* ── TRANSICIONES ── */ .boton { background: #c9a84c; transform: translateY(0); /* propiedad | duración | función | retraso */ transition: all 0.3s ease 0s; /* múltiples propiedades */ transition: transform 0.3s cubic-bezier(.22,1,.36,1), box-shadow 0.3s ease, background 0.2s ease; } .boton:hover { transform: translateY(-4px) scale(1.02); box-shadow: 0 12px 28px rgba(201,168,76,0.4); } /* ── TRANSFORM (se combina con transitions) ── */ .elemento { transform: translateX(20px); /* mover */ transform: translateY(-50%); transform: scale(1.1); /* escalar */ transform: rotate(45deg); /* rotar */ transform: skew(10deg); /* inclinar */ transform: translate(-50%,-50%) scale(1.05); /* combinar */ } /* ── ANIMACIONES KEYFRAMES ── */ @keyframes aparecer { 0% { opacity: 0; transform: translateY(30px); } 100% { opacity: 1; transform: translateY(0); } } @keyframes pulso { 0%, 100% { transform: scale(1); } 50% { transform: scale(1.05); } } @keyframes girar { to { transform: rotate(360deg); } } /* Aplicar animación */ .card { animation: aparecer 0.7s cubic-bezier(.22,1,.36,1) both; } /* Con retrasos escalonados */ .card:nth-child(1) { animation-delay: 0.1s; } .card:nth-child(2) { animation-delay: 0.2s; } .card:nth-child(3) { animation-delay: 0.3s; } /* Propiedades de animación */ .spinner { animation-name: girar; animation-duration: 1s; animation-timing-function: linear; animation-iteration-count: infinite; animation-direction: normal; /* reverse | alternate */ animation-fill-mode: both; /* forwards | backwards | both */ }

Módulo 11 · Diseño adaptable

Responsive Design

El responsive design hace que los sitios web se vean bien en cualquier tamaño de pantalla, desde móviles hasta monitores grandes. CSS3 tiene herramientas nativas para lograrlo.

responsive.css
/* ── MEDIA QUERIES ── */ /* Mobile first: estilos base para móvil, luego se expanden */ /* Base: móvil (< 600px) */ .grid { display: grid; grid-template-columns: 1fr; /* 1 columna en móvil */ gap: 16px; } /* Tablet (≥ 600px) */ @media (min-width: 600px) { .grid { grid-template-columns: repeat(2, 1fr); } } /* Laptop (≥ 900px) */ @media (min-width: 900px) { .grid { grid-template-columns: repeat(3, 1fr); } } /* Desktop (≥ 1200px) */ @media (min-width: 1200px) { .grid { grid-template-columns: repeat(4, 1fr); } } /* ── UNIDADES RELATIVAS ── */ .contenedor { width: 90%; /* % del padre */ max-width: 1200px; margin: 0 auto; padding: 0 1rem; /* rem relativo a root font-size */ font-size: 1em; /* em relativo al padre */ } /* ── VIEWPORT UNITS ── */ .hero { height: 100vh; /* 100% del alto de la ventana */ width: 100vw; /* 100% del ancho de la ventana */ font-size: 5vw; /* 5% del ancho de la ventana */ } /* ── GRID RESPONSIVO SIN MEDIA QUERIES ── */ .auto-grid { display: grid; /* se adapta solo: crea las columnas que quepan */ grid-template-columns: repeat(auto-fill, minmax(280px, 1fr)); gap: 24px; } /* ── PREFERENCIAS DEL SISTEMA ── */ @media (prefers-color-scheme: dark) { /* modo oscuro */ } @media (prefers-reduced-motion: reduce) { * { animation-duration: 0.01ms !important; } /* sin animaciones */ }

Módulo 12 · Profesionalismo

Buenas Prácticas y Ruta

CSS bien escrito es legible, mantenible y eficiente. Estas prácticas separan el CSS amateur del profesional.

✓ Reset global: Siempre inicia con *, *::before, *::after { box-sizing: border-box; margin: 0; padding: 0; }

✓ Variables CSS en :root para colores, espacios y fuentes — nunca valores mágicos dispersos.

✓ Mobile-first: Diseña para móvil primero, luego expande con min-width en media queries.

✓ Nombres de clase descriptivos: .card-header es mejor que .caja-top.

✓ Evita !important — si lo necesitas, hay un problema de especificidad que debes resolver.

✓ Usa unidades relativas (rem, %, vw) en lugar de px fijos para mejor accesibilidad.

✓ Respeta prefers-reduced-motion — no todos quieren animaciones.

Propiedades CSS modernas imprescindibles

gap
Espacio entre items en flex y grid. Reemplaza los margins entre items.
aspect-ratio
Mantiene proporciones. aspect-ratio: 16/9 mantiene el ratio al redimensionar.
container queries
Media queries basadas en el contenedor padre, no en la ventana.
scroll-snap
Crea sliders nativos sin JavaScript con scroll-snap-type y scroll-snap-align.
:is() / :where()
Agrupa selectores: :is(h1, h2, h3) { } Reduce repetición de código.
backdrop-filter
Efecto glassmorphism: blur en el fondo detrás del elemento.
color-mix()
Mezcla dos colores: color-mix(in oklch, blue 30%, white).
@layer
Organiza la cascada en capas explícitas para mejor control de especificidad.
