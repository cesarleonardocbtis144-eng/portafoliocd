Categoría 01

Desarrollo Web

HTML
Marcado 1993

HTML 5

HyperText Markup Language. El esqueleto de toda página web. Define la estructura y el significado semántico del contenido mediante etiquetas.

index.html
<!-- Estructura básica --> <article class="card"> <h1>Cultura Digital</h1> <p>Aprendiendo <strong>HTML</strong></p> <img src="logo.png" alt="Logo"/> </article>
CSS
Estilos 1996

CSS 3

Cascading Style Sheets. El lenguaje que viste a las páginas web: colores, tipografías, animaciones, layouts responsivos y efectos visuales.

styles.css
/* Tarjeta con animación */ .card { display: flex; border-radius: 16px; background: linear-gradient( 135deg, #0f2347, #07111f ); transition: transform .3s ease; }
JS
JS
Interacción 1995

JavaScript

El lenguaje de la interactividad web. Manipula el DOM, gestiona eventos, realiza peticiones al servidor y permite crear experiencias dinámicas.

app.js
// Función con flecha moderna const saludar = (nombre) => { const msg = `¡Hola, ${nombre}!`; console.log(msg); return msg; }; saludar("Cultura Digital");
PHP
PHP
Backend 1994

PHP 8

Hypertext Preprocessor. Lenguaje de servidor que genera HTML dinámico. Impulsa más del 75% de la web, incluyendo WordPress y Facebook en sus inicios.

index.php
<?php // Clase con tipado estricto class Usuario { public function saludar( string $nombre ): string { return "Hola, $nombre"; } } ?>
SQL
SQL
Base de datos 1974

SQL

Structured Query Language. Lenguaje estándar para gestionar bases de datos relacionales. Toda aplicación web usa SQL en MySQL, PostgreSQL o SQLite.

query.sql
-- Consulta con JOIN SELECT u.nombre, c.titulo AS curso FROM usuarios u JOIN cursos c ON u.id = c.usuario_id WHERE u.activo = 1;
TS
TS
Tipado 2012

TypeScript

JavaScript con superpoderes. Agrega tipado estático al código, detecta errores antes de ejecutar y escala proyectos grandes con mayor seguridad.

main.ts
// Interfaz y tipo genérico interface Tarea { id: number; titulo: string; completada: boolean; } const tarea: Tarea = { id: 1, titulo: "Cultura Digital", completada: false, };

Categoría 02

Desarrollo de Escritorio

Py
General 1991

Python

El lenguaje más versátil del mundo. Ideal para IA, ciencia de datos, automatización y escritorio (con Tkinter o PyQt). Sintaxis limpia y legible.

app.py
# Clase con herencia class Estudiante(Persona): def __init__(self, nombre): super().__init__(nombre) self.materias = [] def agregar(self, materia): self.materias.append(materia) return f"✓ {materia}"
Java
Orientado a objetos 1995

Java

Escribe una vez, ejecuta en cualquier lugar. Lenguaje robusto y multiplataforma, base de Android y de grandes sistemas empresariales.

Main.java
// Clase principal public class Main { public static void main( String[] args ) { String msg = "Cultura Digital"; System.out.println(msg); } }
C#
C#
Microsoft 2000

C Sharp

El lenguaje estrella de Microsoft y Unity. Potente para aplicaciones de escritorio con .NET, videojuegos y desarrollo multiplataforma con MAUI.

Program.cs
// Record inmutable (C# 9+) record Alumno( string Nombre, int Semestre ); var a = new Alumno( "María", 2 ); Console.WriteLine(a.Nombre);

Categoría 03

Diseño & Maquetación

Fig
Figma
UI / UX 2016

Figma

La herramienta de diseño colaborativo más popular del mundo. Permite crear interfaces, prototipos interactivos y sistemas de diseño en la nube.

tokens.json (Design Tokens)
{ "color": { "primary": { "value": "#c9a84c", "type": "color" }, "background": { "value": "#07111f" } } }
SCSS
SCSS
Preprocesador 2006

SASS / SCSS

CSS con superpoderes: variables, anidamiento, mixins y funciones. Compila a CSS estándar y hace que los estilos grandes sean mantenibles.

styles.scss
// Variables y anidamiento $gold: #c9a84c; $navy: #07111f; .card { background: $navy; &:hover { border-color: $gold; transform: translateY(-8px); } }
TW
Framework CSS 2017

Tailwind CSS

CSS utility-first. En lugar de escribir clases personalizadas, se aplican utilidades directamente en el HTML para un diseño rápido y consistente.

card.html
<!-- Utilidades directamente --> <div class=" flex flex-col gap-4 rounded-2xl p-6 bg-navy border border-gold/20 hover:-translate-y-2 transition-all "> Contenido de la card </div>
XML
XML
Marcado de datos 1998

XML

eXtensible Markup Language. Lenguaje de marcado para estructurar datos de forma legible. Base de SVG, RSS, Office y muchos formatos de intercambio.

datos.xml
<?xml version="1.0"?> <portafolio semestre="2"> <tarea id="1"> <titulo>Tim Berners-Lee</titulo> <unidad>1</unidad> </tarea> </portafolio>
MD
Documentación 2004

Markdown

Lenguaje de marcado ligero para escribir documentos con formato usando texto plano. Base de README en GitHub, blogs técnicos y plataformas como Notion.
